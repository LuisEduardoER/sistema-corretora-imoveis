
Sistema Corretora Imóveis
Histórias 

Usuarios :
	
	
 	Título: Logar Usuario
 	Um funcionário irá acessar a tela de login de funcionário para ter acesso ao sistema.
 	
 	Com o objetivo de logar o  funcionário.
 	 
 	Dados a Inserir: 
	Login: (E-mail) obrigatório
	Senha: obrigatório. 
	
	--------------------------------------------------------
	
	Título: Alterar Senha Usuario
 	Um funcionário irá acessar a tela de login de funcionário para ter acesso ao sistema.
 	Ira ter uma opçao de trocar a senha.
 	
 	Com o objetivo de Alterar a senha do  funcionário.
 	 
 	Dados a Inserir: 
	Login:(E-mail) obrigatório.
	Senha: obrigatório.
	NovaSenha: obrigatorio.
	
	Dependências: Validar Senha Usuario.
	
	---------------------------------------------------------
	
	Título: Validar Senha Usuario
	
 	O sistema ira procurar nos dados do usuario a senha informada.
 	
 	Com o objetivo de Validar a senha do  funcionário.
 	
 	Dados a inserir:
 	Login:(E-mail) obrigatorio
 	Senha: obrigatorio.
 	 
	Dependências: Alterar Senha Usuario
	
	
	--------------------------------------------------------- 	
 	
 	
 	Título 1: Inserir funcionário
 	Um funcionário já cadastrado  irá acessar a tela de cadastro de funcionário para informar os dados do funcionário a ser cadastrado.
 	
 	Com o objetivo de cadastrar o funcionário.
 	 
 	Dados a Inserir: 

	Nome Completo: obrigatório
	Endereço: obrigatório
	CEP
	Bairro
	Cidade
	Estado
	Email: obrigatório
	Telefone: obrigatório
	CPF: obrigatório
	Data nascimento: obrigatório
	Sexo: M ou F
	Tipo Funcionário: obrigatório
	Login: obrigatório
	Senha: obrigatório. 
	
 	 
 	Dependências: Logar Usuario.
 				  Validar Cpf.
 	 



	Titulo 2: Validar Cpf
	
	O cpf inserido sera submetido a uma validaçao pelo sistema.
	
	Com o objetivo de validar o cpf.
	
	Dados a inserir:
	
	Cpf: obrigatorio.
	
	Dependencias: Logar Usuario
				  Inserir Funcionario.
				  	


 	Título 3: Pesquisar Funcionários 
 	Um funcionário já cadastrado realiza uma busca prévia de funcionários no sistema, sendo utilizados para a pesquisa os seguintes dados: nome, endereço, CPF e tipo de funcionário.

 	Com o objetivo de pesquisar um funcionário já cadastrado.
 	
 	Dados a Exibir: 	
	Nome Completo
	Endereço
	CEP
	Bairro
	Cidade
	Estado
	Email
	Telefone
	CPF
	Data Nascimento
	Sexo
	Tipo Funcionário
 	 
 	Dependências: Usuário logado
 	              Inserir Funcionario
 	              validar cpf



 	Título 4: Alterar Funcionários
 	Um funcionário já cadastrado realiza uma busca prévia de funcionários no sistema, encontrado o funcionário pesquisado, o funcionário poderá fazer a alteração necessária no cadastro.
 	 
 	Com o objetivo de fazer alteração no cadastro de funcionários.
 	
 	Dados a Alterar: 
	Nome completo
	Endereço
	CEP
	Bairro
	Cidade
	Estado
	Email
	Telefone
	CPF
	Data nascimento
	Sexo
	Tipo Funcionário
 	 
 	Dependências: 	Logar Usuario 
 					Inserir Funcionarios
 					Validar Cpf
 					Pesquisar Funcionários
 	 


 	Título 5: Excluir Funcionários

 					Validar Cpf
 					Pesquisar Funcionarios
