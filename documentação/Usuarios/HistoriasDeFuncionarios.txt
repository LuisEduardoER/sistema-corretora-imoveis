
	Sistema Corretora Imóveis
	Histórias 
	
	Funcionário:

 	Título: Logar Funcionário
 	Um funcionário irá acessar a tela de login de funcionário para ter acesso ao sistema.
 	
 	Com o objetivo de logar o  funcionário.
 	 
 	Dados a Inserir: 
	Nome usuário:obrigatório
	Senha: obrigatório. 
 	 
-----------------------------------------------------------------------------------------------

 	Título: Inserir funcionário
 	Um funcionário já cadastrado  irá acessar a tela de cadastro de funcionário para informar os dados do funcionário a ser cadastrado.
 	
 	Com o objetivo de cadastrar o funcionário.
 	 
 	Dados a Inserir: 

	Nome Completo: obrigatório
	Endereço: obrigatório
	CEP
	Bairro
	Cidade
	Estado
	Email: obrigatório
	Telefone: obrigatório
	CPF: obrigatório
	Data nascimento: obrigatório
	Sexo: M ou F
	Tipo Funcionário: obrigatório
 	 
 	Dependências: O funcionário deve estar  logado. 
 	 
	------------------------------------------------------------------------------

 	Título: Pesquisar Funcionários 
 	Um funcionário já cadastrado realiza uma busca prévia de funcionários no sistema, sendo utilizados para a pesquisa os seguintes dados: nome, endereço, CPF e tipo de funcionário.

 	Com o objetivo de pesquisar um funcionário já cadastrado.
 	
 	Dados a Exibir: 	
	Nome Completo
	Endereço
	CEP
	Bairro
	Cidade
	Estado
	Email
	Telefone
	CPF
	Data Nascimento
	Sexo
	Tipo Funcionário
 	 
 	Dependências: Inserir Funcionário
 	 
	------------------------------------------------------------------------------

 	Título: Alterar Funcionários
 	Um funcionário já cadastrado realiza uma busca prévia de funcionários no sistema, encontrado o funcionário pesquisado, o funcionário poderá fazer a alteração necessária no cadastro.

 	Dados a Alterar: 
	Nome completo
	Endereço
	CEP
	Bairro
	Cidade
	Estado
	Email
	Telefone
	CPF
	Data nascimento
	Sexo
	Tipo Funcionário
 	 
 	Dependências: Pesquisar Funcionários
 	 
	------------------------------------------------------------------------------

 	Título: Excluir Funcionários
 	Um funcionário já cadastrado realiza uma busca prévia de funcionários no sistema, encontrado o funcionário pesquisado, o funcionário poderá fazer a exclusão do funcionário no cadastro.

	Endereço

	Bairro
	Cidade

	Email
	Telefone
	CPF

	Sexo
	Tipo Funcionário
 	 
 	Dependências: Pesquisar Funcionários
 	 
	------------------------------------------------------------------------------

 	Título: Alterar Senha Funcionário
	Um Funcionário irá acessar a tela de login de Funcionário para ter acesso ao sistema.
	Ira ter uma opçao de trocar a senha.
	
	Com o objetivo de Alterar a senha do Funcionário.
	
	Dados a Inserir:
	Login:(E-mail) obrigatório.
	Senha: obrigatório.
	NovaSenha: obrigatorio.
	
	Dependências: Inserir Funcionário.
	Validar Cpf
	Validar E-mail
	Logar Funcionário
	Validar Senha Funcionário.
	Validar E-Mail Funcionário
