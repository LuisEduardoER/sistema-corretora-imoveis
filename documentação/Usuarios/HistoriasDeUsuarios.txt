
Sistema Corretora Imóveis
Histórias De Usuários

Usuarios :
	
	
 	Título: Logar Usuario
 	Um usuario irá acessar a tela de login de usuario para ter acesso ao sistema.
 	
 	Com o objetivo de logar o  usuario.
 	 
 	Dados a Inserir: 
	Login: (E-mail) obrigatório
	Senha: obrigatório. 
	
	Dependencias: Inserir Usuario
				  Validar cpf
				  Validar E-mail 
	
	--------------------------------------------------------
	
	Título: Deslogar Usuario
 	
 	Um usuario logado podera a qualquer momento fazer o logout do sistema.
 	
 	Com o objetivo de deslogar o  usuario.
 	 
 	Dependencias: Logar Usuario
 				  Inserir Usuario
 				  Validar Cpf
 				  Validar Email 
 				  	
	--------------------------------------------------------
	
	Título: Alterar Senha Usuario
 	Um usuario irá acessar a tela de login de usuario para ter acesso ao sistema.
 	Ira ter uma opçao de trocar a senha.
 	
 	Com o objetivo de Alterar a senha do  usuario.
 	 
 	Dados a Inserir: 
	Login:(E-mail) obrigatório.
	Senha: obrigatório.
	NovaSenha: obrigatorio.
	
	Dependências: Inserir Usuario.
				  Validar Cpf
				  Validar E-mail
				  Logar Usuario
				  Validar Senha Usuario.
				  Validar E-Mail usuario
	
	---------------------------------------------------------
	
	Título: Validar Senha Usuario
	
 	O sistema ira procurar nos dados do usuario a senha informada.
 	
 	Com o objetivo de Validar a senha do  funcionário já cadastrado.
 	
 	Dados a inserir:
 	
 	Senha: obrigatorio.
 	 
	Dependências: 	Inserir Usuario
					Validar cpf
					Validar Email
					Logar Usuario
					Alterar Senha Usuario
	
	
	---------------------------------------------------------
	Título: Validar E-mail Usuario
	
 	O sistema ira procurar nos dados do usuario a E-mail informado.
 	
 	Com o objetivo de Validar o Email do  funcionário já cadastrado.
 	
 	Dados a inserir:
 	
 	E-mail: obrigatorio.
 	 
	Dependências: 	Inserir Usuario
					Validar cpf
					Validar E-mail
					Logar Usuario
					Alterar Senha Usuario	
	---------------------------------------------------------
 	
 	
 	Título 1: Inserir Usuario
 	
 	Um usuario já cadastrado  irá acessar a tela de cadastro de usuario para informar os dados do usuario a ser inserido.
 	
 	Com o objetivo de cadastrar o usuario.
 	 
 	Dados a Inserir: 

	Nome Completo: obrigatório
	Endereço: obrigatório
	CEP : obrigatório
	Bairro : obrigatório
	Cidade : obrigatório
	Estado : obrigatório
	Email: obrigatório
	Telefone: obrigatório
	CPF: obrigatório
	Data nascimento: obrigatório
	Sexo: M ou F : obrigatório
	Tipo Funcionário: obrigatório
	Login:(E-mail) obrigatorio
	Senha: obrigatório. 
	
 	 
 	Dependências: Logar Usuario.
 				  Validar Cpf.
 				  Validar E-mail
 				  
	-----------------------------------------------------------------

	Titulo 2: Validar Cpf
	
	O cpf inserido sera submetido a uma validaçao pelo sistema.
	
	Com o objetivo de validar o cpf.
	
	Dados a inserir:
	
	Cpf: obrigatorio.
	
	Dependencias: Logar Usuario
				  Inserir Usuario.
				  
	----------------------------------------------------------------	
	Titulo 3 : Validar E-mail
	
	O E-mail inserido sera submetido a uma validaçao pelo sistema.
	
	Com o objetivo de validar o E-mail.
	
	Dados a inserir:
	
	E-mail: obrigatorio.
	
	Dependencias: Logar Usuario
				  Inserir Usuario.

			  	

	-----------------------------------------------------------------------------------------------------
 	Título 4: Pesquisar Usuarios 
 	Um usuario já cadastrado realiza uma busca prévia de usuarios no sistema, sendo utilizados para a pesquisa os seguintes dados: nome, endereço, CPF e tipo de usuario.

 	Com o objetivo de pesquisar um usuario já cadastrado.
 	
 	Dados a Exibir: 	
	Nome Completo
	Endereço
	CEP
	Bairro
	Cidade
	Estado
	Email
	Telefone
	CPF
	Data Nascimento
	Sexo
	Tipo Funcionário
 	 
 	Dependências: Usuário logado
 	              Inserir Usuario
 	              Validar cpf
 	              Validar E-mail


	--------------------------------------------------------------------------------------------
 	Título 5: Alterar Usuario
 	Um usuario já cadastrado realiza uma busca prévia de usuarios no sistema, encontrado o usuario pesquisado, o usuario poderá fazer a alteração necessária no cadastro.
 	 
 	Com o objetivo de fazer alteração no cadastro de usuarios.
 	
 	Dados a Alterar: 
	Nome completo
	Endereço
	CEP
	Bairro
	Cidade
	Estado
	Email
	Telefone
	CPF
	Data nascimento
	Sexo
	Tipo Funcionário
 	 
 	Dependências: 	Logar Usuario 
 					Inserir Usuarios
 					Validar Cpf
 					Validar E-mail
 					Pesquisar Usuarios

	CPF

	Tipo Funcionário
 	 
 	Dependências:	Logar usuario
 					Inserir Usuario
 					Validar Cpf
 					Validar E-mail
 					Pesquisar Usuario
